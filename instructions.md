# Assignment 1: Sentiment Analysis with k-Nearest Neighbor Classifier

## Overview and Assignment Goals

The objectives of this assignment are:

- Implement the Nearest Neighbor Classification Algorithm
- Handle Text Data (Movie Reviews)
- Design and Engineer Features from Text Data
- Choose the Best Model (Parameters of Nearest Neighbor Selection, Features, and Similarity Functions)

## Detailed Description

### Task

Your task is to infer sentiment (or polarity) from free-form review text submitted for movies.

You are required to implement a k-Nearest Neighbor Classifier to predict sentiment for 25,000 reviews in the test file (`test.dat`):
- **Positive sentiment** = +1 rating
- **Negative sentiment** = -1 rating

In `test.dat`, you are only provided the reviews without ground truth ratings (which will be used for evaluation).

### Data Format

**Training Data** (`train.dat`):
- 25,000 reviews
- Each row: sentiment score followed by review text
- Each review ends with `#EOF` to denote the end

**Test Data** (`test.dat`):
- 25,000 reviews (no sentiment scores provided)
- Same format as training data with `#EOF` markers

**Output Format** (`format.txt`):
- Example file with 25,000 rows alternating with +1 and -1
- Your `output.txt` should follow this format with sentiment scores generated by your model

### Evaluation Metrics

Accuracy will be used to compare your predictions on the test set with the ground truth (hidden).

**Important notes:**
- The public leaderboard shows results for 50% of randomly chosen test instances only (standard practice to avoid gaming)
- The private leaderboard will be used for final ranking
- You may submit predictions **10 times per 24-hour cycle**
- Cross-validate diligently before submitting
- Final ranking is based on the submission with the highest score
